# Game Jam Brief Generator

A small, client-side tool that generates short game jam briefs and themes.

Features
- Generate randomized briefs from themes listed in `themes.json`.
- Single-file web app: open `index.html` in a browser.

Usage
- Open [index.html](index.html) in your browser.
- Or serve locally:

	- Python 3: `python -m http.server 8000` and open `http://localhost:8000`

Development
- Edit the files and reload the page. No build step required.
